## Списки

Чтобы добавить нумерованные списки, необходимо выделить пункты звездочкой (*).
Например вот так:
* первый элемент.
* второй элемент.

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт.
2. второй пункт.
